A simple QR generator for Haiku

Aug 12, 2021 by Fabio Tomat aka TmTFx

This little piece of code written in python is a simple frontend that lets you generate QRs in Haiku and BeOS.
I hope you enjoy it even if it has a lot of limits.
The program uses some python libraries to create QRs, add a logo and, at your will, save it wherever you want;
you can create QRs of links or simple text, you can nest a QR within a QR using the logo feature.
This program has been written to share links with cell phones.
