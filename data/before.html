Before running...

I used python version 2.7.18 and the program needs some modules installed:

  • Bethon: a collection of Python modules for BeOS/Haiku; I used version 0.5.4.1 from my git repo

  • qrcode: python module to create qrs

  • pillow: python module to handle images

  • backports.tempfile: a backport from python3 of tempfile module to handle temporary files
